Uses

Make sure to check out uses.tech for a list of everyone's /uses pages!

I will try to keep things updated, so this page will serve as a living document and a place to point curious developers to when I get asked.

    Editor & IDE

  • PhpStorm after half a year with NetBeans it's my IDE of choice since over 5 years
  • Sublime Text is my editor for single files - I use it often to edit Kodi-NFO, JSON, bash and others.
  • WakaTime beside several other plugins this is one of my most used to track my project activity
  • Nord is my theme of choice since some time.
  • Fira Code is my font of choice with enabled ligatures.

    Terminal

  • Terminal & bash - yes, I use the default MacOS Terminal app with bash.
  • Starship is my prompt of choice since some time.
  • Nord is my theme of choice since some time.
  • Fira Code is my font of choice with enabled ligatures.
  • I have all my Dot files in a public GitHub repo.

    Apps

  • I use Glyphfinder and Rocket to find emojis and glyphs.
  • As a G-Suite user I stick to Backup and Sync and Drive File Stream to access all my cloud files, sync them and backup my local folders.
  • I've tried Alfred some time ago and now use it as an awesome replacement for Spotlight
  • To manage my movies, music and so on I use FileBot combined with tinyMediaManager.
  • To have a proper window manager I use Rectangle.
  • FontBase is the first tool I've found that's capable to manage/display multiple thousand fonts.
  • I'm using Umbra to toggle between light and dark theme with a simple shortcut.

    Hardware

  • I'm using a MacBook Pro (15-inch, 2018) as my primary computer.
  • On my desk I have two DELL Ultrasharp U2515H as my screens.
  • I'm using a wired Apple Keyboard but waiting for german layout of the Keychron K1.
  • Since ~10 years my favorite mouse is a Logitech M500
  • On the go a Zendure Supertank powers my MacBook and phone.
  • As headset I'm using a Jabra Elite 85h.
  • Since >3 years I'm using a Huawei P10 but it has to be replaced in near future - think that I will go for a Google Pixel 4a.
